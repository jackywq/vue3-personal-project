# 人事管理系统 (HR System)

基于 Vue3 + Element Plus + Pinia 构建的现代化人事管理平台。项目遵循模块化、可扩展、低耦合的设计原则，采用标准的 5 层分层架构，覆盖人事管理的核心全流程。

## 🌟 核心特性

- **现代技术栈**：采用 Vue 3 (Composition API) + Vite 极速构建。
- **UI 组件库**：集成 Element Plus，提供一致且美观的视觉交互体验。
- **状态管理**：使用 Pinia 进行全局及模块化状态管理，职责清晰。
- **路由控制**：基于 Vue Router 的动态路由和权限拦截守卫。
- **通用组件**：高度封装了 `BaseTable`（表格）、`BaseSearch`（搜索）和 `BaseForm`（表单）等高频业务组件，大幅降低重复代码。
- **接口规范**：统一的 Axios 请求封装，内置 Token 携带与全局异常处理机制。

## 📁 项目结构

项目采用严谨的分层架构设计：

```text
hr-system/
├── src/
│   ├── api/          # 接口请求封装 (拦截器、业务模块接口)
│   ├── assets/       # 静态资源 (图片、全局样式)
│   ├── components/   # 全局通用组件 (BaseTable, BaseSearch, BaseForm)
│   ├── layouts/      # 布局组件 (主布局包含侧边栏、头部)
│   ├── pinia/        # 状态管理 (按模块划分，如 user.js)
│   ├── router/       # 路由配置 (路由守卫与模块化路由)
│   ├── utils/        # 通用工具函数
│   └── views/        # 核心业务页面
│       ├── login/        # 登录页
│       ├── dashboard/    # 仪表盘
│       ├── employee/     # 员工管理模块
│       ├── dept/         # 部门管理模块
│       └── attendance/   # 考勤管理模块
```

## 🚀 快速开始

### 环境要求

- Node.js >= 18.0.0
- npm 或 pnpm

### 安装依赖

```bash
npm install
```

### 本地开发

```bash
npm run dev
```

启动后，访问 `http://localhost:5173/`。

> **测试账号**
> 用户名: `admin`
> 密码: `123456`

### 生产构建

```bash
npm run build
```

## 🛠️ 主要业务模块

1. **工作台 (Dashboard)**：展示员工总数、入职统计、缺勤情况等核心数据指标。
2. **员工管理**：提供员工列表查询、状态过滤，支持员工的新增、编辑和删除。
3. **部门管理**：管理组织架构，查看各部门的编码、人数及负责人信息。
4. **考勤管理**：记录员工的每日打卡状态（正常、迟到、缺勤等）。

## 💡 设计原则

- **基础层**：Vite 构建、Axios 网络请求。
- **核心层**：Vue 3、Pinia、Vue Router 提供全局支撑。
- **业务层**：按业务域拆分逻辑，解耦不同模块。
- **展示层**：依托 Element Plus 与自定义组件统一交互。
